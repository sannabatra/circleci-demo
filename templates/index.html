🚀 Welcome to My DevOps Pipelineeee!

This project showcases a simple CI/CD pipeline built with CircleCI.

📦 Technologies Used:

  • Python3🐍
  • Flask
  • Pytest 🧪
  • CircleCI
  • Git & GitHub

✅ What It Does:

  1. Runs tests on every commit to main
  2. Ensures the Flask app is working correctly
  3. Provides automated feedback via CircleCI pipeline
